Cypher is a privacy-preserving prediction market protocol built on Solana. Unlike traditional prediction markets where every bet is visible the moment it's placed, Cypher encrypts your chosen side client-side using x25519 and RescueCipher — so the network sees your stake, but never your prediction. When a market closes, Arcium's Multi-Party Computation network decrypts and settles all positions in a single tamper-proof computation, then pays out winners automatically in USDC.

## How it works

Getting from zero to a placed bet takes three steps. The SDK handles the encryption and MPC orchestration for you — you just initialize a client, find a market, and call `placeBet`.

<Steps>
  <Step title="Install the SDK">
    Add `@cypher-zk/sdk` and Solana web3.js v1 to your project.

    ```bash
    npm install @cypher-zk/sdk @solana/web3.js@^1.95.4
    ```
  </Step>
  <Step title="Connect your wallet">
    Create a `CypherClient` with your RPC connection and wallet adapter. The constructor is synchronous — no async setup required.

    ```typescript
    import { CypherClient } from "@cypher-zk/sdk";
    import { Connection } from "@solana/web3.js";

    const client = new CypherClient({
      connection: new Connection("https://api.devnet.solana.com"),
      wallet,
      cluster: "devnet",
    });
    ```
  </Step>
  <Step title="Place an encrypted bet">
    Call `client.actions.placeBet` with your chosen market and side. The SDK encrypts your prediction before it ever touches Solana.

    ```typescript
    const { sig, userKeypair } = await client.actions.placeBet({
      marketId: 1n,
      side: 1,          // 1 = YES, 0 = NO
      betAmountUsdc: 5_000_000n,  // $5 USDC
    });

    // ⚠️ Persist userKeypair.privateKey — it cannot be recovered on-chain
    localStorage.setItem(
      `secret:${marketId}`,
      Buffer.from(userKeypair.privateKey).toString("hex"),
    );
    ```
  </Step>
</Steps>
